# 🤖 Virtual Girlfriend Bot

Telegram бот с ИИ персонажами на базе Groq AI (LLaMA 3.3).

## ✨ Особенности

- 👥 **4 уникальных персонажей** с разными характерами
- 🧠 **Groq AI Integration** - мощная модель LLaMA 3.3
- ⚡ **Бесплатный** - используется бесплатный API Groq
- 🎯 **Интерактивные кнопки** - удобная навигация
- 🔄 **Смена персонажей** - переключайтесь между девушками
- 💬 **Естественные ответы** - AI понимает контекст

## 🚀 Быстрый старт

### 1️⃣ Клонируем репозиторий

```bash
git clone https://github.com/dimonel2898-design/virtual-girlfriend-bot.git
cd virtual-girlfriend-bot
```

### 2️⃣ Устанавливаем зависимости

```bash
pip install -r requirements.txt
```

### 3️⃣ Получаем API ключи

#### Telegram Bot Token:
1. Откройте Telegram и найдите [@BotFather](https://t.me/botfather)
2. Напишите `/newbot`
3. Следуйте инструкциям и получите **BOT_TOKEN**

#### Groq API Key:
1. Зайдите на [console.groq.com](https://console.groq.com)
2. Авторизуйтесь
3. Скопируйте **API Key**

### 4️⃣ Создаем .env файл

```bash
cp .env.example .env
```

Отредактируйте `.env` и добавьте ваши ключи:

```
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 5️⃣ Запускаем бота

```bash
python main.py
```

Вы должны увидеть:
```
🤖 Бот запущен с Groq AI!
```

## ⚙️ Конфигурация

### Изменение моделей AI

В файле `ai_responses.py` найдите строку:
```python
model="llama-3.3-70b-versatile",
```

Доступные модели Groq:
- `llama-3.3-70b-versatile` - мощная модель (по умолчанию)
- `mixtral-8x7b-32768` - быстрая модель
- `gemma-7b-it` - легкая модель

### Кастомизация персонажей

Отредактируйте словарь `CHARACTERS` в `main.py` для изменения персонажей.

Отредактируйте `prompts` в `ai_responses.py` для изменения поведения AI.

## 🐛 Решение проблем

### "ModuleNotFoundError: No module named 'groq'"
```bash
pip install groq
```

### "Invalid BOT_TOKEN"
Проверьте, что вы скопировали правильный токен из @BotFather

### "Invalid GROQ_API_KEY"
Убедитесь, что API ключ активирован на [console.groq.com](https://console.groq.com)

### "Бот не отвечает"
1. Убедитесь, что скрипт запущен: `python main.py`
2. Проверьте логи на ошибки
3. Перезагрузите бота
